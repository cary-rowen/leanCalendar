# 简易万年历（leanCalendar）

简易万年历（leanCalendar）是一款 NVDA 插件，用于快速读出农历、节气和公历日期时间信息，也可打开查询对话框查看其他日期。

## 命令

* `NVDA+F11`：读出农历和节气信息。
* 连按两次 `NVDA+F11`：读出更多农历和黄历相关上下文。
* 连按三次 `NVDA+F11`：打开简易万年历查询对话框。
* 输入手势中另有一个未分配手势的命令，可直接打开简易万年历查询对话框。
* `NVDA+F12`：读出当前公历时间。
* 连按两次 `NVDA+F12`：读出公历日期详情。

## 各命令读出内容

### NVDA+F11

按一次读出：

* 农历日期。
* 农历月大小。
* 年、月、日、时干支。
* 当前节气第几天。
* 当前候。
* 农历节日（如有）。
* 下个节气倒计时，仅在 7 天内读出。

连按两次读出：

* 附近节气。
* 距下个节气的时间。
* 生肖。
* 望月时间。
* 月相。
* 日十二建。
* 三伏、数九、梅雨等传统时段（如适用）。

连按三次打开查询对话框。可按公历日期时间、农历日期时辰或节气查询；结果按公历、农历、节气物候、黄历分组显示。

### NVDA+F12

按一次读出当前时间，精确到秒。

连按两次读出：

* 公历日期和星期。
* 今年第几天、第几周、第几季度、当月天数。
* 假日、法定休息日或调休上班状态（如可用）。
* 非假日时的工作日或周末状态。
* 星座。

农历和节气数据由内置的 [tyme4py](https://github.com/6tail/tyme4py) 库计算。

## 许可证

简易万年历基于 GNU 通用公共许可证第 2 版发布。
